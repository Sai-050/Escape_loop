# 🚪 Escape Loop

Escape Loop is an AI-powered quiz game where players solve coding and logic questions to progress through unlimited levels. The game uses Google Gemini AI to generate unique questions, making every playthrough different and challenging.

## Features

- Infinite levels with increasing difficulty
- AI-generated coding and logic questions
- Categories like Debugging, DSA, Big O, Tech Trivia, and Cybersecurity
- Score tracking and level progression
- Firebase-based data storage
- Retro hacker terminal UI
- Fast and responsive gameplay

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Google Gemini API
- Firebase Firestore
